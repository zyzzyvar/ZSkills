# 三时段分析方法论

## 通用分析顺序(每只股票)
1. 先看大盘与板块环境(覆水之下无完卵,大盘弱时个股强势难持续)
2. 资金面 → 技术面 → 基本面 → 消息面,四面交叉验证
3. 当多个维度指向一致时信号强;矛盾时,短线信资金面,中长线信基本面+政策

## 盘前分析(定调)
- 大盘:`fetch.py market` 看昨日主力净流入、北向态度、指数位置;判断今日情绪偏暖/偏冷
- 个股:对每只 `snapshot`,重点看
  - 昨日主力是否净流入、是否放量(量比>1.5 需关注资金意图)
  - 隔夜 `news` 有无重大消息(利好/利空/政策)
  - 价格位置:距MA20、MA60的关系,关键支撑压力位
- 产出:当日"剧本"——预期情绪、每票关注点、触发动作的价位

## 午间分析(校验)
- 上午量价 vs 盘前预期:符合则维持计划,背离则警觉
- 盘中异动识别:
  - 放量上涨但涨幅收窄 → 上方抛压重,警惕诱多
  - 缩量回调不破支撑 → 多为洗盘,持股
  - 突放巨量 → 查是否有消息,无消息的天量往往是变盘信号
- 关注股:是否触及预设的介入价位/条件

## 盘后分析(复盘)
- 全天资金定性:主力净流入是真实建仓还是对倒诱多?结合换手率与龙虎榜席位判断
- 北向当日加减仓方向(聪明钱参考)
- 持仓盈亏:对照成本价,逐票评估持有逻辑是否仍成立
- 明日预判:支撑压力、可能的主力意图、操作建议
- 关注股:是否进入买入区间,给出明确的触发条件

## 关键指标解读速查
- 量比:>1 放量,>2 明显放量,<0.8 缩量。放量上涨健康,放量下跌危险
- 换手率:3-7% 活跃健康;>10% 过热(高位危险/低位可能启动);<1% 关注度低
- MACD:DIF上穿DEA金叉偏多,水上金叉强于水下;高位死叉减仓信号
- 均线:多头排列(MA5>MA20>MA60)趋势向上;跌破MA20警惕走弱
- 主力净流入:连续流入是建仓/拉升前兆;价涨钱出是出货嫌疑
