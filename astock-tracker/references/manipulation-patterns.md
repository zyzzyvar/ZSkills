# 主力手法识别图谱

A股散户占比高,主力(机构/游资)惯用手法操纵情绪。识别手法是实战核心。
核心原则:价格可以做假,资金流和筹码相对难做假;手法识别要"量价资"三者结合。

## 诱多(骗散户追进,主力出货)
特征:
- 高位放量假突破:突破前高但量能不持续,次日即回落
- 尾盘急拉:临近收盘拉升做出好看K线,次日高开诱多后低走
- 利好兑现下跌:重大利好公布当天高开,但主力借机出货,放量滞涨/长上影
识别:股价新高但主力资金净流出、北向减仓 → 拉高出货嫌疑,不追
应对:不追高,持仓者考虑减仓

## 洗盘(甩掉浮筹,后续还要拉)
特征:
- 缩量回踩:下跌但量能萎缩,不破关键支撑(MA20/前低)
- 急跌快速收回:盘中插针后拉回,制造恐慌
- 横盘磨人:长时间窄幅震荡消磨耐心
识别:回调缩量、主力资金未大幅流出、关键支撑不破 → 多为洗盘
应对:持股,可在支撑位附近低吸;跌破支撑且放量则改判

## 出货(主力派发离场)
特征:
- 高位放量滞涨:量大但价格不再创新高
- 天量长上影/巨阴:见顶信号
- 连续净流出:主力资金持续流出但股价靠对倒维持
识别:量价背离 + 主力持续净流出 + 北向减仓
应对:止盈离场,不抱幻想

## 吸筹/建仓(底部主力进场,后续可能拉升)
特征:
- 底部缩量企稳,偶有放量但回落有限
- 主力资金悄悄净流入,股价不涨或缓涨(不希望被发现)
- 北向/机构席位龙虎榜出现
识别:底部区域主力持续净流入 + 缩量 → 建仓嫌疑,可关注
应对:可逢低分批介入,设好止损

## 重要提醒
- 单一信号不足为凭,必须多维度交叉。例如"缩量回踩"既可能是洗盘也可能是
  下跌中继,要结合大盘、基本面、主力资金综合判断
- 主力手法在演变,且会"反向利用"散户的技术分析惯性。保持怀疑,以资金面为锚
- 永远设止损。判断错了,止损是唯一的保护
